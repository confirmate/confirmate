# Agent Instructions

This document holds guidance for automated agents (like Copilot) interacting with the
Confirmate repository. It replaces the previous instructions that lived in
`.github/copilot-instructions.md`.

- When adding or modifying tests, read and follow [CONTRIBUTING.md](CONTRIBUTING.md)
  (especially the **Testing Guidelines** section) before implementing changes.
  The repository contains explicit unit test style rules that are checked by CI,
  so make sure your test code adheres to them.
